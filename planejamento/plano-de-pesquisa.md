Título do Projeto:
Desenvolvimento de Sistema Web para Gestão e Comunicação do Grêmio Estudantil

Categoria:
(    ) Ciências da Natureza e Exatas
( X ) Informática
(    ) Ciências Humanas e Linguagens
(    ) Engenharias

Resumo:
Este projeto tem como objetivo desenvolver um sistema web para o grêmio estudantil, com foco na melhoria da comunicação interna, organização de documentos e interação com os alunos. A proposta surge da necessidade de centralizar informações, já que atualmente muitos dados estão dispersos em diferentes meios, dificultando o acesso e a organização. O projeto será desenvolvido com base em estudos da área de desenvolvimento de sistemas e organização de dados, utilizando tecnologias web. Espera-se como resultado a criação de uma plataforma funcional que facilite a gestão do grêmio, melhore a comunicação com os alunos e contribua para uma participação mais ativa da comunidade escolar.

Palavras-Chave:
Sistema Web, Grêmio Estudantil, Comunicação Digital, Desenvolvimento de Software, Organização de Dados

Plano de pesquisa

Introdução:
O uso da tecnologia no ambiente escolar tem crescido significativamente, sendo uma ferramenta importante para melhorar a comunicação e a organização de informações. No caso do grêmio estudantil, muitas atividades dependem de uma boa comunicação e acesso rápido a documentos e avisos. No entanto, atualmente essas informações estão espalhadas em diferentes plataformas, como redes sociais e mensagens, o que dificulta o acesso e a organização.

Esse cenário pode gerar problemas como falhas na comunicação, perda de informações e baixa participação dos alunos. Diante disso, surge a necessidade de desenvolver um sistema que centralize todas essas informações em um único ambiente digital.

Assim, este projeto propõe a criação de um sistema web que auxilie na organização do grêmio estudantil, facilitando a comunicação interna e externa, além de melhorar a gestão das informações.

Objetivos:

Objetivo Geral
Desenvolver um sistema web para melhorar a comunicação, organização e gestão do grêmio estudantil.

Objetivos Específicos

* Levantar os requisitos do sistema com base nas necessidades do grêmio
* Planejar a estrutura e funcionalidades do sistema
* Desenvolver a interface do sistema
* Implementar funcionalidades como avisos e organização de documentos
* Realizar testes e ajustes no sistema

Metodologia:
O projeto será desenvolvido com base em pesquisa aplicada, pois busca resolver um problema real dentro do ambiente escolar. Também será utilizada pesquisa bibliográfica, com base em materiais sobre desenvolvimento de sistemas e organização de dados.

Inicialmente, será realizado um levantamento de requisitos para identificar as necessidades do grêmio. Em seguida, será feito o planejamento do sistema e o desenvolvimento utilizando tecnologias web, como HTML, CSS e JavaScript.

Após o desenvolvimento, serão realizados testes para verificar o funcionamento do sistema e possíveis melhorias. Caso necessário, poderão ser aplicados questionários com alunos para validar a solução.

Cronograma:

* Mês 1/2 – Planejamento e levantamento de requisitos
* Mês 3 – Desenvolvimento inicial
* Mês 4/5 – Implementação de funcionalidades
* Mês 6/7 – Testes e finalização

Resultados Esperados:
Espera-se desenvolver um sistema web funcional que permita organizar as atividades do grêmio estudantil de forma eficiente. O sistema deverá melhorar a comunicação com os alunos, facilitar o acesso a documentos e organizar as informações internas.

Além disso, espera-se aumentar a participação dos alunos nas atividades do grêmio e melhorar a gestão das informações.
